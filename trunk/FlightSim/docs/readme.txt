	The project covers all requirements specified by the documentation.  Here are all keys usable:

	p/P : pause movement
	//? : toggle filling
	./> : toggle lighting
	g/G : toggle fullscreen

	i/I : set initial position and velocity

	o/O : toggle time of day proceeding

	f/F : move forward
	a/A : roll left
	d/D : roll right
	w/W : increase velocity
	s/S : decrease velocity

	c/C : toggle cockpit

	q/Q : quit program
