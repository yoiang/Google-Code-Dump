Ian G
Particles

rotate camera view: hold left mouse button and drag mouse
zoom camera view: hold right mouse button and drag mouse
key mappings:
	esc :	quit program
	=/+:	add particles
	-/_:	remove particles
	g/G:	toggle gravity
	w/W:	toggle wind
	l/L:	toggle "connection the dots" =P
	c/C:	remove all particles
	t/T:	enable textures
	s/S:	cycle textures
	1/!:	Demonstrate Fountain type spawner
	2/@:	Toggle Snowfall presets
	3/#:	Toggle Fire presets
